# Bingo_Generator


## 概要
BINGOの抽選機

## 説明
このアプリはウェブベースのビンゴゲーム抽選機です。

NEXT BALLボタンを押すごとにランダムで数字が抽選され、抽選された数字は表に埋められていきます。

NEW GAMEボタンを押すとリセットされ、1から始められます。
